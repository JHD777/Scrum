Eventos de Scrum

Scrum prescribe cuatro eventos formales para la inspección y adaptación, contenidos dentro de un quinto evento: el Sprint.

SPRINT
1-4 Semanas
Planning
Max 8h

El equipo colabora para definir el objetivo del Sprint y planificar cómo lograrlo. Se selecciona qué se hará y cómo.

Daily Scrum
15 min/día

Evento diario para los Developers. Inspeccionan el progreso hacia el Sprint Goal y adaptan el Sprint Backlog.

Retrospective
Max 3h

El equipo se inspecciona a sí mismo (personas, procesos, herramientas) para crear un plan de mejoras para el próximo Sprint.

Sprint Review
Max 4h

El equipo y los stakeholders revisan el resultado del Sprint. Se inspecciona el Incremento y se adapta el Product Backlog.
